Off-Grid Power System Overhaul: 1980 Bluebird Wanderlodge

This interactive guide details a Victron-focused approach to overhauling the off-grid power system for a 1980 Bluebird Wanderlodge RV. It covers project goals, system benefits, understanding the existing setup, and a high-level view of the proposed new system.

Project Goals and Scope

The primary objective is to design and detail the installation of a robust and reliable electrical system, prioritizing Victron Energy components. Air conditioning units are excluded from the off-grid electrical loads. The project aims for a DIY installation within a two-day timeframe, requiring meticulous preparation.

Benefits of Victron-Based System

Opting for Victron Energy offers enhanced reliability, advanced features like PowerAssist and UPS functionality, integrated monitoring via Cerbo GX and VRM portal, scalability, and strong community support. This system aims to provide dependable off-grid power and enhance self-sufficiency.

High-Level Diagram of New Power System

The conceptual flow of power is designed for efficiency and reliability:

Power Flow:

  1. Solar Panels → Solar Combiner Box (if needed) → Victron SmartSolar MPPT Charge Controller
  2. MPPT Controller → Main DC Fuse & Disconnect Switch → Victron Lynx Distributor (or equivalent) → Battery Bank (LiFePO4)
  3. From Lynx Distributor / Busbars:
    • → Victron MultiPlus-II Inverter/Charger → RV AC Distribution Panel (120V AC loads)
    • → RV DC Fuse Panel (12V DC loads)
  4. The MultiPlus-II also accepts AC input (Shore/Generator) for passthrough and battery charging.

The original system includes a 12V DC supply (four 6V batteries), two 45A converter/chargers (to be replaced by MultiPlus-II), an AC supply system with selector switch, and DC monitoring. Understanding these is crucial for decommissioning.

System Component Selection & Sizing (Victron Focused)

This section details the recommended Victron components and their sizing based on the revised power needs.

A. Battery Bank (LiFePO4)

LiFePO4 batteries are recommended for their long cycle life, deep discharge capability, lighter weight, and safety.

Required Capacity: Based on 205 Ah daily need, 2 days autonomy, and 80% max DoD: (205 Ah * 2 days) / 0.80 = 512.5 Ah. Target: 515-600 Ah.

The report highlights a potential "Opening Restriction: 24"L x 5"W". This is a MAJOR constraint. The recommended SoK SK12V314PH (17"L x 9.4"W x 10.9"H) and similar models WILL NOT FIT if this 5" width is accurate. Action Required: Meticulously re-measure the smallest access point (width AND height) to your battery compartment. This is the absolute first step before purchasing any batteries. If the 5" restriction is firm, you'll need to find multiple smaller LiFePO4 batteries that can fit or consider relocating the entire battery bank.

Middletown, CA averages ~4.5 Peak Sun Hours (PSH) conservatively. Required Solar Watts = (205 Ah * 12.8V) / 4.5 PSH / 0.80 efficiency = ~729W. Aim for 800W-1200W for resilience.

Recommendation: Three 380W-400W monocrystalline panels (1140W-1200W total). Mount securely to roof stringers, sealing all penetrations with Dicor 501LS. Consider adjustable tilt mounts for better winter harvest.

  • Branch Circuits: Reuse existing 12V DC panel if suitable, fed from Lynx.

Use appropriate gauges (e.g., 4/0 AWG for battery/inverter, 10 AWG PV wire, 2 AWG for MPPT output, 6 AWG for MultiPlus AC). Use high-quality copper lugs (hydraulic crimped) and MC4 connectors (properly crimped). Adhesive-lined heat shrink is essential.

Installation: Step-by-Step (Target: 2 Days)

This intensive schedule assumes "Day 0" prep is complete and all materials/tools are ready.

Day 0: Crucial Pre-Work (To be done BEFORE the 2-day window)

  1. Safety First: ABSOLUTELY NO WORK ON LIVE CIRCUITS. Disconnect ALL power (shore, generator, old batteries - NEGATIVE FIRST). Cover any existing solar. PPE on. Fire extinguisher ready.
  2. Review Existing System: Familiarize with 1980 Wanderlodge manual. Locate old batteries, converters, AC selector, breaker/fuse panels.
  3. Identify & Label Wiring: Before disconnecting, trace and label ALL wires from old components. Take many photos.
  4. Safely Remove Old Gear: Remove old batteries. Disconnect AC input and DC output from old converters. Remove converter units. Reuse existing DC panel if good.
  5. Clean & Prep Mounting Locations: Clean battery compartment. Clean surfaces for new components. Ensure structural soundness and ventilation.
  6. Plan Component Layout & Wire Runs: Use actual components or mock-ups. Place them, plan cable routes (shortest practical), measure accurately. This is key for the 2-day goal.

Day 1: Component Mounting and Heavy Gauge Wiring

Morning (Solar Panel Installation - Approx. 4-5 hours)

  1. Prepare Roof Surface: Clean thoroughly where mounts and cable gland go.
  2. Install Solar Panel Mounts: Locate roof stringers. Secure mounts to them if possible. If Z-brackets, attach to panels first. If tilt, assemble and attach bases. Critical Sealing: Dicor 501LS under mount feet AND over all fasteners/perimeters.
  3. Install Roof Cable Entry Gland: Choose location for drip loop and short internal run. Drill hole(s). Seal gland thoroughly with Dicor.
  4. Mount Solar Panels: Carefully lift panels (2-person job). Secure to mounts.
  5. Run PV Wires: Route from panels to gland. Make MC4 connections if needed on roof. Pass wires through gland. Seal gland openings. Route wires towards MPPT (DO NOT connect yet).

Afternoon (Battery and Major DC Component Installation - Approx. 3-4 hours)

  1. Install Batteries: Address compartment opening restriction! Place and secure batteries firmly.
  2. Mount Major DC Components: MultiPlus-II (ventilated, near batteries), SmartSolar MPPT (cool, ventilated, near PV entry & batteries), Lynx Distributor (central), SmartShunt (main negative path), Disconnect Switch & Class T Fuse Holder (main positive path, near battery).
  3. Wire Main DC System (Heavy Gauge): Use correct lugs, hydraulic crimps, heat shrink. Torque to spec.
    • Battery Interconnects (if parallel): Equal length/gauge cables.
    • Battery Positive Path: Batt Main Pos → Class T Fuse → Disconnect Switch → Lynx Pos Input.
    • Battery Negative Path: Batt Main Neg → SmartShunt "Batt Minus" → SmartShunt "System Minus" → Lynx Neg Input.
    • Inverter to Lynx: MultiPlus DC Pos → Fused Lynx Pos. MultiPlus DC Neg → Lynx Neg. (4/0 AWG or 2x 1/0 AWG).
    • MPPT to Lynx: MPPT BAT+ → Fused Lynx Pos. MPPT BAT- → Lynx Neg. (2 AWG or 1/0 AWG for 100A MPPT).
  4. Connect Solar PV Wires to MPPT: Once MPPT is connected to battery bus, connect PV array wires to MPPT PV+/PV-. CRITICAL: Ensure correct polarity.

Day 2: AC Wiring, Monitoring, and Final Connections

Morning (AC Wiring and Integration - Approx. 3-4 hours)

  1. MultiPlus-II AC Wiring: All AC power OFF.
    • AC Input (AC-IN): Route 6 AWG (for 50A) from RV AC source selector/ATS output → MultiPlus AC-IN (L, N, G).
    • AC Output (AC-OUT1): Route 6 AWG from MultiPlus AC-OUT1 (L, N, G) → RV main AC panel input.
    • Decommission Old Converter AC Feed: Turn OFF breaker, disconnect wire, make safe.
    • Grounding: MultiPlus chassis ground lug → RV chassis. Ensure AC panel ground bar also bonded to chassis.
  2. Verify AC Panel for Inverter Loads: Confirm all intended circuits are fed by MultiPlus AC-OUT1.

Afternoon (Monitoring, Low Voltage DC, and Final Checks - Approx. 3-4 hours)

  1. Install Monitoring System: Mount Cerbo GX, GX Touch. Connect MultiPlus (VE.Bus), MPPT (VE.Direct/Can), SmartShunt (VE.Direct) to Cerbo. Connect SoK CANbus (if applicable). Power Cerbo & Touch.
  2. Wire Existing 12V DC Loads: Connect original 12V DC panel feed wire to a fused output on Lynx.
  3. Final Wiring Checks: Meticulously double-check ALL connections for tightness & polarity. Inspect wire routing for protection. Confirm correct fuses installed.
  4. Tidy Up & Labeling: Organize wiring (loom/conduit). Clearly label components, fuses, key wires.

System Testing and Commissioning

Systematic testing ensures correct and safe operation.

A. Initial Power-Up Sequence

  1. Final Visual Inspection: Check all wiring, connections, no loose tools.
  2. Power Up Batteries: Main disconnect OFF. DC load breakers OFF. Turn ON main disconnect. Check SmartShunt/Cerbo for battery voltage.
  3. Power Up DC System: Turn ON DC breakers one by one. Test each DC load.
  4. Power Up MPPT: With sunlight, MPPT should power on. Via VictronConnect/GX Touch, verify it sees battery voltage, PV voltage/current.
  5. Power Up Inverter/Charger (MultiPlus-II): No AC input connected. Turn MultiPlus to "Inverter Only" or "On". Listen for start. Test AC outlet with small load. Check Cerbo/VictronConnect for status.

B. Verifying Voltages and Polarities

Use DMM to check DC voltage and polarity at key points (battery, Lynx, MPPT, MultiPlus DC inputs). Check AC output voltage (~120V AC).

C. Testing Solar Charging

With sunlight and MPPT configured, observe charging current. Verify MPPT progresses through LiFePO4 charge stages (Bulk, Absorption, Float).

D. Testing Inverter AC Output Under Load

On inverter power, test AC appliances. Monitor battery voltage, inverter load, battery discharge current. Do not exceed inverter rating.

E. Testing Shore Power Passthrough and Charging

  1. MultiPlus to "Charger Only" or "On".
  2. Connect shore power/start generator.
  3. MultiPlus should detect AC input, pass through to AC panel.
  4. Verify AC loads work from shore/gen.
  5. Verify MultiPlus is charging batteries. Monitor current, voltage, stage.

F. Configuring Victron Components

‚öôÔ∏è

Configuration is Critical for LiFePO4!

Use VictronConnect or GX Touch. Incorrect settings can damage batteries or reduce lifespan.

  • Battery Settings (MultiPlus & MPPT): Correct total capacity, "LiFePO4" chemistry, precise charge voltages (Bulk/Absorb, Float) per SoK specs. If CANbus connected, BMS may control this.
  • MultiPlus Settings: Input current limits for shore, PowerAssist, UPS function.
  • SmartShunt Settings: Match battery capacity.
  • Cerbo GX Settings: Network for VRM, ensure devices recognized.

G. Monitoring System Performance

Operate under normal conditions for hours. Monitor key parameters. Check for overheating. Address issues immediately.

Additional Information and Best Practices

Key considerations for safety, maintenance, and future planning.

A. Safety Precautions Summary (Reiterate and Emphasize)

‚ùó

Safety is Paramount!

  • Electricity is Dangerous: Consult a qualified professional if uncertain.
  • De-energize All Circuits: Verify OFF with a multimeter.
  • Proper Fusing is Non-Negotiable: Protects wiring and components.
  • Battery Safety: Avoid short circuits. Ensure BMS is functional.
  • Roof Safety: Use caution, consider safety harnesses.

B. System Maintenance Recommendations

  • Connections: Inspect high-current connections for tightness/corrosion every 6-12 months.
  • Solar Panels: Clean regularly to remove dirt/debris.
  • Battery Terminals: Visual inspection.
  • System Monitoring: Regularly check performance via Cerbo/VRM for anomalies.
  • Firmware Updates: Apply Victron updates for optimal performance.

C. Troubleshooting Common Issues

  • No DC Power: Check disconnect, main fuse, battery SoC, connections, DC breakers.
  • No AC Power from Inverter: Check inverter ON, battery SoC/voltage, DC input fuse, fault codes, AC output breakers.
  • Low Solar Charging: Check shading, panel cleanliness, PV wiring, MPPT settings, PV fuse/breaker.
  • Inverter/Charger Fault Codes: Consult MultiPlus manual (low/high battery voltage, overload, over-temp are common).

D. Future Expansion Possibilities

  • Solar Array Expansion: Add panels if space/MPPT capacity allows (150/100 can handle >1200W). May need 2nd MPPT.
  • Battery Bank Expansion: Add identical LiFePO4 batteries in parallel. Ensure charging system can handle it.
  • Other Victron Accessories: DC-DC chargers (Orion-Tr Smart), sensors, etc.
